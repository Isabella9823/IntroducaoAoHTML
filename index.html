Introdução ao HTML


TAGS - Estruturais

<html>...</html> A tag HTML abreviação de HyperText Markup Language, é usada para estruturar o conteúdo de uma página da web.
<head>...</head> A tag HEAD providencia informações gerais sobre o documento, incluindo seu título e links para scripts e folhas de estilos.
<tittle>...</tittle> A tag TITTLE define o título do documento, mostrado na barra de título de um navegador ou na aba da página.
<body>...</body> A tag BODY é usada para definir o corpo de um site em HTML. Tudo que estiver dentro da tag será mostrado de alguma forma no conteúdo da página.

TAGS - Semânticas

<article>...</article> A tag ARTICLE tem como objetivo ser independentemente distribuível ou reutilizável.
<section>...</section> A tag SECTION é um temático agrupamento de conteúdos.
<head>...</head> A tag HEAD providencia informações gerais sobre o documento, incluindo seu título e links para scripts e folhas de estilos.
<aside>...</aside> A tag ASIDE é destinado a conteúdos que não fazem parte do fluxo do texto, mas que, ainda sim, estão relacionados de alguma forma.
<main>...</main> . A tag MAIN diz para os navegadores onde está o conteúdo principal de nossa aplicação
<small>...</small> A tag SMALL sempre aparece dentro de um elemento <footer> ou <aside>, normalmente incluindo informações de direitos autorais ou avisos legais, além de outras "letras miúdas".

TAGS - Texto

<p>...</p> A tag P Principal tag de texto, compõe um parágrafo.
<b>...</b> A tag B é um temático agrupamento de conteúdos.
<i>...</i> A tag I Transforma o conteúdo em itálico.
<br>...</br> A tag BR não necessita de fechamento, ela executa a função de quebra de linha.
<hr>...</hr> . A tag HR não necessita de fechamento, ela forma uma linha horizontal.
<span>...</span> Apesar de ter uma funcionalidade e características parecidas com os parágrafos, costumam ser utilizadas apenas para pequenas informações, como legendas de um formulário, legendas de uma imagem, etc. Também pode ser utilizada para formar um container.

TAGS - Imagens

<img>...</img> A tag IMG representa a inserção de imagem no documento.
src A tag B é essencial para carregar recursos embutidos em elementos HTML, como imagens, áudio, vídeo, scripts e iframes..
alt A tag ALT Especifica um texto alternativo para a imagem, se a imagem por algum motivo não puder ser exibida.
<map >...</map > A tag MAP é usado com os elementos para definir um mapa de imagem.
<area>...</area> A tag AREA elemento define uma região hot-spot em uma imagem.
<picture>...<picture> A tag PICTURE é um container usado para especificar múltiplos elementos <source> para um elemento específico <img> contido nele.

TAGS - Link

<a>...</a> A tag A com o atributo href cria-se um hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL.
</link> A tag LINK especifica onde abrir o documento vinculado por uma tag a.
<target>...</target> A tag TARGET especifica onde abrir o documento vinculado por uma tag a.
